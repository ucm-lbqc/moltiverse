# moltiverse

Moltiverse is a command-line application for generating molecule conformers.

## Installation

```
shards install
```

## Requirements
External software:
- [OpenBabel](https://openbabel.org)
- [Namd](https://www.ks.uiuc.edu/Research/namd/)
- [Ambertools](https://ambermd.org/AmberTools.php)

## Usage
